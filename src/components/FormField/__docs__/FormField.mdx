# FormField

A text input field component supporting labels, required indicators, loading, warning, error, and disabled states. The FormField component is designed for flexible form usage and clear state feedback.

## Examples

<Canvas of={FormFieldStories.AllVariants} />

## Usage

```tsx
import { FormField } from '@femmecubator/prima-ds';

// Basic usage
<FormField label="Name" placeholder="Enter your name" />

// Required field
<FormField label="Email" required placeholder="Enter your email" />

// Loading state
<FormField label="Loading" loading placeholder="Loading..." />

// Disabled state
<FormField label="Disabled" disabled placeholder="Can't edit" />

// Warning state
<FormField label="Username" warning placeholder="Choose a username" />

// Inline warning
<FormField label="Username" warningInline="Username is taken" placeholder="Choose a username" />

// Error state
<FormField label="Password" error placeholder="Enter password" />

// Inline error
<FormField label="Password" errorInline="Password is too short" placeholder="Enter password" />

// Controlled usage
const [value, setValue] = useState('');
<FormField label="Controlled" value={value} onChange={e => setValue(e.target.value)} />
```
