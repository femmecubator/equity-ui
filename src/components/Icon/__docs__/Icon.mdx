A versatile icon component that displays SVG icons from a sprite with customizable sizes and theme-based colors.

## Usage

```tsx
import { Icon } from '@aroris/prima-ds';

// Basic usage
<Icon name="settings" />

// Custom size
<Icon name="email" size="small" />
<Icon name="menu" size={32} />

// With theme colors
<Icon name="success-circle" color="success-icon" />
<Icon name="error-triangle" color="error-icon" />

// All props
<Icon
  name="warning-circle"
  size="large"
  color="warning-icon"
/>
```
